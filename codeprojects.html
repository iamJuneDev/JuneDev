Bubble Sort

The quickest and most efficient way in sorting a set of data can be obtained through a QuickSort algorithm when choosing an element from set array and comparing value with every other element in the array pivot. The result places all elements greater than the pivot into one subgroup and all elements that are less than into another subgroup. Each subgroup is divided until it only contains one item and then all items are sorted in order.
